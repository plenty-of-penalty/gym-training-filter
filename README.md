# gym-training-filter

> 队长为了给队员加训用心良苦

* 2017~2018 赛季及之后的场次
* ICPC 赛制
* 时长为 5 小时
* 难度至少为 4☆
* 有中文或英文题解
* 有至少一个 friends 做过
